# Salesforce Admin Exam Prep

Interactive exam preparation website for the **Salesforce Certified Administrator (ADM-201)** exam.

## Features

- Topic guides for all 8 exam domains
- Interactive flashcard practice (Q&A and MCQ)
- Trailhead links for each topic
- Cheat sheet / quick reference
- Netlify-ready deployment

## Deploy to Netlify

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/IbrahimIF/salesforce-admin-prep)

## Local Development

Just open `index.html` in your browser — no build step required.

## Tech Stack

- Plain HTML, CSS, JavaScript
- No frameworks or dependencies
- Fully static — works on Netlify, GitHub Pages, or any CDN
